
# 📐 Phase 1: The Foundations

> **Duration**: 3–4 weeks · **Difficulty**: 🟢 Beginner · **Prerequisites**: None

Build the mathematical intuition and Python fluency that every AI concept builds upon.

---

## Notebooks

| # | Notebook | Topic | Key Concepts |
|:-:|----------|-------|--------------|
| 1 | [`01-linear-algebra-for-ai.ipynb`](01-linear-algebra-for-ai.ipynb) | **Linear Algebra** | Vectors, matrices, tensors, eigenvectors, matrix multiplication |

