# 📊 Phase 2: ML Fundamentals

> **Duration**: 3–4 weeks · **Difficulty**: 🟡 Intermediate · **Prerequisites**: Phase 1

Master classical machine learning — still the best choice for structured tabular data, and the foundation for understanding deep learning.

| # | Notebook | Topic | Key Concepts |
