## Phase 4: Generative AI & LLM Engineering

### Prompt Engineering

| Resource | Type | Tag | Description |
|----------|:----:|:---:|-------------|
| [DeepLearning.AI — ChatGPT Prompt Engineering](https://www.deeplearning.ai/short-courses/chatgpt-prompt-engineering-for-developers/) | 🎥 | ⭐ | Andrew Ng's concise intro to prompt engineering. Free. |

### Fine-Tuning & Alignment

| Resource | Type | Tag | Description |
|----------|:----:|:---:|-------------|
| [HuggingFace TRL Documentation](https://huggingface.co/docs/trl/) | 🔗 | ⭐ | The standard library for fine-tuning and alignment (SFT, DPO, PPO). |
| [HuggingFace PEFT Documentation](https://huggingface.co/docs/peft/) | 🔗 | ⭐ | Parameter-Efficient Fine-Tuning library (LoRA, QLoRA). |
| [`hiyouga/LLaMA-Factory`](https://github.com/hiyouga/LLaMA-Factory) | 💻 | 📎 | All-in-one fine-tuning toolkit. Great for experimentation. |
| [LoRA (Hu et al., 2021)](https://arxiv.org/abs/2106.09685) | 📄 | ⭐ | The original LoRA paper. Understand how parameter-efficient tuning works. |
| [DPO (Rafailov et al., 2023)](https://arxiv.org/abs/2305.18290) | 📄 | ⭐ | Direct Preference Optimization — the 2026 standard for alignment. |
| [QLoRA (Dettmers et al., 2023)](https://arxiv.org/abs/2305.14314) | 📄 | 📎 | Fine-tune 65B models on a single GPU. |

## Recommended Books

### Essential Reading (Start Here)

| Book | Author | Why Read It |
|------|--------|-------------|
| *Build a Large Language Model (From Scratch)* | Sebastian Raschka | Hands-on LLM implementation. The best way to truly understand GPT. |
| *Designing Machine Learning Systems* | Chip Huyen | The definitive guide to production ML systems and MLOps. |
| *Hands-On Large Language Models* | Jay Alammar, Maarten Grootendorst | Practical LLM engineering with beautiful illustrations. |

### Deep Dives

| Book | Author | Why Read It |
|------|--------|-------------|
| *Deep Learning* | Goodfellow, Bengio, Courville | The "bible" of deep learning theory. Free online. |
| *Hands-On Machine Learning* (3rd ed.) | Aurélien Géron | Classical ML + deep learning with scikit-learn and PyTorch. |
| *An Introduction to Statistical Learning* | James, Witten, et al. | Statistical foundations of ML. Free online. |
| *Natural Language Processing with Transformers* | Lewis Tunstall et al. | HuggingFace Transformers in depth. |
